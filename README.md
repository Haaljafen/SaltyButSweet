# Salty But Sweet 🧁
A 3D time-management game where you run a bakery and serve customers before their patience runs out.

## Gameplay
- Prepare food at stations and place them on the counter
- Serve the correct order to each customer before they leave
- Earn money to get stars — the more you earn, the better your rating
- Manage your time wisely across 3.5 minutes of bakery chaos

## Controls
| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move |
| E | Prepare food / Serve customer |
| R | Trash plate |
| Escape | Pause |

## Scoring
| Stars | Money Required |
|-------|---------------|
| ⭐ | $50+ |
| ⭐⭐ | $100+ |
| ⭐⭐⭐ | $150+ |
